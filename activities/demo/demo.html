What is the difference between block tags and inline tags.

  • Block tags starts a new line and take up the full width available.
  • Inline tags do not start a new line and take up as much width as necessary.

What is the margin, border, and padding?

  • Margin is space outside the element, creating distance between it and other elements.
  • Border is a line that surrounds an element.
  • Padding is space inside an element, between the content and its border.
