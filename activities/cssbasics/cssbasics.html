CSS Basics

Inline css

Inline CSS is applied directly after paragraph. p style

Embedded CSS

Embeded styles are placed inside a style tag within the head section of the HTML file

External CSS

External CSS styles are defined in a separate css file and linked to the HTML file using a link tag.

Why you should only use external CSS?

External CSS is easier to manage and update style across multiple pages and most updated method.

What is a selector used for?

Selector is used to choose which HTML elements to style. It changes the color of the elements.
